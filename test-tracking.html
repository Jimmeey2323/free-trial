Integration Code Examples

JavaScript Fetch Example

// Get all leads fetch('/api/leads') .then(response => response.json()) .then(data => console.log(data)); // Get campaign statistics fetch('/api/campaigns/stats') .then(response => response.json()) .then(stats => { stats.forEach(campaign => { console.log(`${campaign.campaign}: ${campaign.count} leads`); }); });

Python Example

import requests # Get all leads response = requests.get('https://your-app.up.railway.app/api/leads') leads = response.json() # Filter by source google_leads = [l for l in leads if l['utm_source'] == 'google'] print(f"Google leads: {len(google_leads)}")
